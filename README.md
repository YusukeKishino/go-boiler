# go-boiler
boiler plate for web application with golang

# Requirement
* goenv
* nodenv

# Installation
```bash
go get -u github.com/YusukeKishino/go-boiler
```

# Usage
```bash
go-boiler -n app-name -p github.com/YusukeKishino -g 1.15.2 --node-version 14.5.0 -o path/to/output
```
